如何不翻墙安装chrome插件

2017-02-08

最近发现仅仅通过修改 hosts 文件翻墙的话，可以上 google，但是还是没法上 chrome 商店，不知道为什么。所以就查了一下如何不翻墙安装 chrome 插件，在 segmentDefault 上找到了一篇文章。

segmentDefault 原文地址：https://segmentfault.com/q/1010000002574198

这篇文章的第一个回答比较方便，就是通过 http://chrome-extension-downloader.com/ 这个网站下载 chrome 插件。在这个网站下载插件需要知道插件的 url 或者 id，那么如何才能知道一个插件的 url 或者 id 呢？你在 bing 里搜索这个插件，比如我要下载 axure chrome 插件的话，我就可以在 bing 里搜 axure chrome extension 关键词，第一个就是这个插件的 chrome id，然后复制这个网址，找到最后一个 / 之后的内容，即 dogkpdfcklifaemcdfbildhcofnopogp，然后将这部分内容复制到上面提供的网址的输入框，点击 download 就可以下载了。插件如何安装就不说了，自行搜索。)

总结一下：bing 搜索插件，获得插件网址，尽管无法打开这个网址，但是可以知道这个插件的网址或者 id，然后将 id 输入到 http://chrome-extension-downloader.com/ 这个网站，点击下载就可以了。
