Gist, Gitbook and Gitlab

这段时间在 github 上看到了很多的好项目，觉得 github 真是个不错的地方。但是在 github 最上面的导航里一直有一个 gist 的链接没有点进去过。后来点进去之后发现是一个很简单的空页面，不知道干什么的，然后就开始了解 gist，后来发现这个 gist 简直是太好用了，可以用来记录一些零碎的代码片段，或者是当做一个自己的备忘来用，而除了这些还有很大的用处。这个需要专门做一个专题来讲。

除了 gist 之外，后来还发现一个 gitbook，里面也是很多不错的内容，好像是可以根据你的 github 仓库来形成一本书，而这个书是可以导出来成 pdf 以及其他很多格式的。这个也需要单独来写一个专题。

后来又发现一个叫 gitlab 的东西，结果发现这个是和 github 类似，但是完全是两个公司在运营的东西，有相似的地方，也有很大的不同。有一篇文章讲解的不错。
